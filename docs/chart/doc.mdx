## Usage

```tsx
import { Chart } from "@/components/chart";
```
```tsx
const data = [12, 18, 15, 24, 22, 30, 28, 35, 42, 38, 48, 56];

<Chart data={data} />
```

The chart auto-scales the Y axis to the data range. Hover snaps to the nearest data point.
